Semantic Elements

<header>

通常包含 logo / 主標題 / 網站描述 / <nav>（<header>的導航欄不要到別人的網站）

<nav>

主要跟"連結"相關，不限於只能在<header>裡

<main>

主要內容，可以包含多個<section>

此元素每個頁面使用一次就好。不要放重複的內容(例如header/footer/aside已經有的內容)

<section>

將內容以主題作分段，通常帶有標題

<article>

通常是獨立的文章，含標題 / 內容等等

<aside>

內容應該要與主要內容間接相關，可以是廣告 / 文章 / 連結

<strong>跟<em>

強調用，取代以前的<b>跟<i>

<figure>

通常是獨立的內容，影片 / 圖片 / 備註等等

備註會放在<figcaption>裡面

<blockquote>

引用了其他來源的東西，包含書籍 / 文章 / 名言等等

引用的作者會放在<cite>裡面

<footer>

通常包含 作者資訊 / 版權資訊 / 聯絡資訊 / <nav>（<footer>的導航欄通常是到其他個人社群）

自訂html屬性

自訂的屬性通常命名為data-name，name可以自訂

想要利用js選取特定元素時會用到，例如document.querySelector('element[data-name]')，如果想指定自訂屬性的value可以用document.querySelector('element[data-name]=value')，value可以是變量也可以是字串
