CSS & Css tricks

一些可能很常用，但以前不熟悉的東西

pointer-events當某內容position: fixed;覆蓋整個畫面時，想點擊其他內容點不到時，可以設置pointer-events: none;讓覆蓋畫面的點擊事件無效，就可以點擊覆蓋在下方的其他內容了

user-select當不想讓人選取到文本時，可以設置user-select: none;

overflow可以控制超出容器外的content該如何設定，要隱藏內容還是添加滑動來觀看所有content等

aspect-ratio可以控制圖片或容器的長寬比例

animation的shorthand需要與keyframes一起用才能產生動畫效果

object-fit跟background-size一樣，只是這個是對img元素進行操作

skeleton loading css

預先做好要加載畫面的容器大小，並且加載後利用js將其隱藏或移除

除了作出容器，還需要讓容器要animation的效果，讓用戶知道畫面正在跑動

一般會在容器的偽類::before或::after上做特效，而非容器本身，容器本身主要是決定寬高，設定postion: relative來作為偽類的基本定位

偽類基本設定一定包含，繼承容器寬高(width和height可以設100%)，使偽類存在的content: "";，使偽類定位在容器上方的position: absolute;及z-index，特效記得用漸層透明的做法，再加animation移動漸層位置達到效果

特效別設在容器內的東西，否則東西長寬不同，會導致動畫特效不一致

加載圖片要用div包住，div要先預設大小，否則會有page-jumping的問題

預防圖片有image-jumping的問題，div最好預設postion: relative;及overflow: hidden;img則要設置position: absolute;

當內容沒有超出範圍，讓<footer>貼齊下方空間的方法

content in the <body> without <footer>{
    min-height: calc(100vh - <footer>height)
}

rwd，偵測手機、平板橫向模式的顯示

@media (orientation: portrait) {
    content...
    }

portrait是指手機等的橫向模式

利用flex輕鬆將navbar的內容標題分成左右邊的方法

.container{
    display: flex;
}

    margin-right: auto;
}
