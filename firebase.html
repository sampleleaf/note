如何使用firebase

部屬react到firebase

firebase官網

步驟1：點選get started

步驟2：建立專案

步驟3：在firebase控制台點選"建構"，點選HOSTING

步驟4：安裝firebase cli，npm install -g firebase-tools

步驟5：登入google，firebase login

步驟6：啟動專案，firebase init

步驟7：輸入指令之後，會問要選擇哪個feature，用spacebar勾選，那一般前端選這個Configure files for Firebase Hosting and (optionally) set up GitHub Action deploys

步驟8：firebase會要連結到哪個專案，現有還是新創等等...

步驟9：如果是react記得是要連到build，而不是public

步驟9.5：建立的firebase檔案建議放在.gitignore，別上傳到gitrepo

步驟10：再來跑react的npm run build

步驟11：部屬至firebase託管，firebase deploy
