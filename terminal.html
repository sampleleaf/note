  • [command] --help

    man [command] （for mac users）
    指令的操作手冊
  • vi
    打開vi編輯器（若內建有vim會使用vim）
    vi [file]
    打開檔案，此時在"命令模式"(若無檔案會創建檔案)
    按i進入"插件模式"，才能開始編輯
    按esc回到"命令模式"
    按:w保存文件再按:q退出文件（也可以直接按:wq保存退出）
  • nano
    打開nano編輯器
    nano [file]
    打開nano編輯器編輯檔案
  • whoami
    輸出username
  • cat

    cat [file]
    查看檔案的內容(內容太多不會全顯示)
    cat [include]*[include]
    *代表擴展名，include沒輸入，會顯示當前資料夾所有檔案的內容
    顯示包含[include]的檔案的內容（例如"*副檔名","in*.html","*d*"都有效）
    cat -n [file]
    查看檔案的內容，內容每行會列號
  • more

    more [file]
    查看檔案的內容(內容太多會分段輸出)
  • less

    less [file]
    查看檔案的內容(內容一次性輸出，案q退出後內容不會留在terminal視窗)
  • echo

    echo [something]
    輸出"輸入的內容"
  • history
    輸出"執行過的命令歷史記錄"
    history -c
    清除"執行過的命令歷史記錄"
  • cp

    cp [copy-file] [paste]
    複製檔案並貼上
    paste如果是file，則檔案會以paste名稱複製到當前路徑
    paste如果是pathway，則檔案會複製到指定路徑
    cp -r [copy-directory] [paste]
    複製資料夾(含子檔)並貼上
  • mv

    mv [origin-file] [new-file]
    更改檔案名稱
    mv [file/directory] [pathway]
    移動檔案或資料夾位置
  • find
    搜訊當前資料夾所有內含的directory及file
    find -name "[directory/file]"
    尋找特定名稱的資料夾或檔案
  • grep
    grep [string] [file]
    搜尋file中含有特定"字串"的行
  • ls
    顯示當前資料夾的檔案
    ls -a
    顯示檔案(包含隱藏的檔案)
    ls -l
    顯示檔案及其詳細資訊
    ls -t
    顯示檔案(以修改時間做排序)
    ls [pathway]
    顯示指定路徑的檔案
  • clear
    快捷鍵為ctrl+L
    將terminal視窗清乾淨
  • cd [pathway]

    cd ..
    移動到上一個資料夾
    cd /[絕對路徑]
    /是根目錄
    cd ~[路徑]
    ~是當前工作目錄的home directory
  • pwd
    顯示當前資料夾的路徑
  • rm

    rm [file]
    刪除文件
    rm -r [directory]
    刪除資料夾及其內容(會詢問確認是否刪除)
    rm -rf [directory]
    強制刪除資料夾及其內容
  • mkdir

    mkdir [directory]
    創建資料夾(無法含子資料夾)
    mkdir -p [directory]
    遞迴創建資料夾及子資料夾
