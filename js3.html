JsNotion

querySelectorAll選取元素的特殊方法

一般都是抓id抓class抓elements

其實也可以抓attribute selector

範例：document.querySelectorAll('input[type="text"]')或抓自定義的屬性document.querySelector(`a[data-category="monkey"]`)

如何獲取被觸發事件的元素的自定義屬性(data-name)

categoryLink.addEventListener('click', (e) => { console.log(e.target.getAttribute('data-category')); //e.target.data-category dot-notation捕獲不到 })

如果是正常的內建屬性javascript是可以用dot-notation來捕獲的

HTMLcollection和NodeList可否使用array.method

在現代browser，nodeList仍然是類array，但已經可以使用array.method

HTMLcollection則不行，但是利用spread和array轉換就可以。htmlArray = [...HTMLcollection]

模板文字

${expression}可以像這樣${variable || 'string'}，當variable為false時會套用string，反之true則套用variable

短路求值

其實上方模板文字就是利用這個技巧

a&&b，如果a為true則回傳b

a||b，如果a為false則回傳b

JavaScript Window Location

想要透過javascript切換網址，可以使用window.location.href，不過window不一定要打出來，其實用location.href同樣可以直接訪問

new URL()

new URL(相對or絕對網址)可以將網址從string變成可訪問的object

searchParams

new URL(window.location.href).searchParams會返回一個對象，對象的key有properties和method

例如作為properties的size會回傳params的數量

例如作為method的forEach的callback有3個argument，但主要前2個可以回傳url的paramKey和paramValue

獲取paramKeys跟paramValues就可以進行操作，比如fetchAPI的template的${expression}就可以置入

Scroll Event

通常會使用到innerHeight&scrollY&offsetHeight

innerHeight是viewport的高度，可以當作100vh

scrollY是垂直方向滾動的pixel

HTMLElement.offsetHeight會讀取元素的高度，如果要獲取整個page的高度，使用document.body.offsetHeight就可以了

當網頁滾動到底時的條件就是window.innerHeight + window.scrollY >= document.body.offsetHeight

額外補充，當因為用戶螢幕過大，沒有出現滾輪時，可以把條件設置成window.innerHeight === document.body.offsetHeight時觸發，可能會運用到遞迴(ex.在a函數內再次調用a函數)

pushState

如果想更改url，可以用history.pushState(null, null, URL);

URL用相對url或絕對url都可以，但以上方code而言，他只會改變url，不會帶你去其他頁面，除非自己額外加code處理

但就算不去其他頁面，也有更改網頁的方法，如果以抓api呈現在頁面來講，我們可以抓url的參數代入api，重新fetchapi來更改頁面

IIFE

可以立即執行函數，函數沒名稱也可以執行(但不能被呼叫，所以只會執行一次)，實際可應用在fetch上面
