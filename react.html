何謂jsx

jsx是javascript的語法擴展，允許我們編寫類似html的程式碼在js的檔案中

react不會創建實際的DOM節點，創建的html-element不是真正的element，它創建的是一個描述DOM節點的object

可以試著console.log()，就知道答案了

要將jsx渲染到網頁上，我們需要利用react的createRoot API

這個類html標籤內或標籤之間都可以添加{js表達式}，稱為"jsx表達式"

jsx-規則

  1. Return a single root element這個意思是當我們要寫複數元素時，記得要用<div></div>或<></>包住，否則error，因為無法直接傳回多個object

    <div></div>或<></>的區別只是在網頁顯示上用不用div包住而已

  2. Close all the tags所有標籤末尾一定要加/

  3. camelCase就是命名任何名字要保持的格式，例如類html中沒有class，得替換成className

createRoot

作為entry-point的main.jsx通常會預設好一個createRoot，當然我們也可以自己修改

渲染網頁時，index.html中的<div id="root"></div>內的所有內容會被jsx取代，包含這個元素本身，但在這元素以外則不會

創建component

用function來製作component，作為component的function名稱開頭要大寫

function內要用return來回傳我們所輸入的jsx及內容，如果要輸入多行，得用return()，將jsx及內容輸入到括號內。不用括號的話，return那行以下的內容將被忽略

在.render中用jsx標記(大寫的function名稱加一個空格可以作為標記名稱)來顯示component

傳遞props到component

主要功能可以說props將資料從parent-component傳到child-component

作為component的function的第一個參數為props

jsx中的標記所添加的屬性(props)，都可以被調用。例如<Header haha="something"/>，haha就是props的key

注意，props在react是"read-only-value唯獨值"，所以不能直接修改。此為錯誤範例：props.haha = "something2"，這樣會error

useState hook

Data from state is distributed to components through props

這是hook裡面的其中一種功能，用來管理react-component的state(Managing data that changes over time)

useState(初始值)回傳一個陣列，陣列有兩個值，[0]為參數傳入的"初始值"，[1]為可以改變初始值的set函數

我們可以利用react的內建事件onClick，來觸發改變初始值的事件

要改變初始值得在set函數的參數內，初始值無法直接修改，改變初始值，實際上是用新值取代，而非直接改變初始值

const [state, setState] = useState(initialState) state不能直接被更改，他是const。setState是一個函數，函數內的參數會取代state的值，而不是非要以state來更改。

setState函數的參數內建立的函數的參數會是"當前的state值"

batch state update的概念


    const Counter = ({name}) => {
        const [quantity, setQuantity] = useState(0) //記得useState的參數無法直接從外部定義變量傳入，需要利用react內部組件調用，例如props
        
        const incrementQuantity = () => {
            setQuantity(quantity + 1)
            if(name == 'Apples'){
            setQuantity(quantity + 1)
            }
        }
    }

以上述情況，在javascript中當名稱為Apples時，最後結果是會是+2

但在react中state是用replace(取代)的方式，且react會等到事件處理程序中的所有程式碼都執行完畢後才處理狀態更新

所以最後結果Apples仍是+1

如果想要累加的方式必須使用函數，setQuantity(quantity + 1)改成setQuantity(quantity => quantity + 1)，雖然一般不會這樣做
