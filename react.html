何謂jsx

jsx是javascript的語法擴展，允許我們編寫類似html的程式碼在js的檔案中

react不會創建實際的DOM節點，創建的html-element不是真正的element，它創建的是一個描述DOM節點的object

可以試著console.log()，就知道答案了

要將jsx渲染到網頁上，我們需要利用react的createRoot API

這個類html標籤內或標籤之間都可以添加{js表達式}，稱為"jsx表達式"

jsx-規則

  1. Return a single root element這個意思是當我們要寫複數元素時，記得要用<div></div>或<></>包住，否則error，因為無法直接傳回多個object

    <div></div>或<></>的區別只是在網頁顯示上用不用div包住而已

  2. Close all the tags所有標籤末尾一定要加/

  3. camelCase就是命名任何名字要保持的格式，例如類html中沒有class，得替換成className

createRoot

作為entry-point的main.jsx通常會預設好一個createRoot，當然我們也可以自己修改

渲染網頁時，index.html中的<div id="root"></div>內的所有內容會被jsx取代，包含這個元素本身，但在這元素以外則不會
